# Mix_PauseAudio

Suspend or resume the whole audio output.

## Syntax

```c
void Mix_PauseAudio(int pause_on);

```

## Function Parameters

|                  |                                          |
| ---------------- | ---------------------------------------- |
| **pause_on**     | 1 to pause audio output, or 0 to resume. |

## Version

This function is available since SDL_mixer 3.0.0.
