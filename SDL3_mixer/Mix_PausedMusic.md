# Mix_PausedMusic

Query whether the music stream is paused.

## Syntax

```c
int Mix_PausedMusic(void);

```

## Return Value

Return 1 if music is paused, 0 otherwise.

## Version

This function is available since SDL_mixer 3.0.0.
