# SDL_GameControllerGetNumTouchpads

Get the number of touchpads on a game controller.

## Syntax

```c
int SDL_GameControllerGetNumTouchpads(SDL_GameController *gamecontroller);

```

## Version

This function is available since SDL 2.0.14.
