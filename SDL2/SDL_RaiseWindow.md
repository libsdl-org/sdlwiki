# SDL_RaiseWindow

Raise a window above other windows and set the input focus.

## Syntax

```c
void SDL_RaiseWindow(SDL_Window * window);

```

## Function Parameters

|                |                     |
| -------------- | ------------------- |
| **window**     | the window to raise |

## Version

This function is available since SDL 2.0.0.
