# SDL_MAX_UINT8

An unsigned 8-bit integer type.

## Header File

Defined in [SDL_stdinc.h](https://github.com/libsdl-org/SDL/blob/SDL2/include/SDL_stdinc.h)

## Syntax

```c
#define SDL_MAX_UINT8   ((Uint8)0xFF)           /* 255 */
```
