# SDL_JoystickClose

Close a joystick previously opened with [SDL_JoystickOpen](SDL_JoystickOpen.md)().

## Syntax

```c
void SDL_JoystickClose(SDL_Joystick *joystick);

```

## Function Parameters

|                  |                              |
| ---------------- | ---------------------------- |
| **joystick**     | The joystick device to close |

## Version

This function is available since SDL 2.0.0.
