# SDL_GL_DeleteContext

Delete an OpenGL context.

## Syntax

```c
void SDL_GL_DeleteContext(SDL_GLContext context);

```

## Function Parameters

|                 |                                  |
| --------------- | -------------------------------- |
| **context**     | the OpenGL context to be deleted |

## Version

This function is available since SDL 2.0.0.
