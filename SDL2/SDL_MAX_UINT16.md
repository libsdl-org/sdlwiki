# SDL_MAX_UINT16

An unsigned 16-bit integer type.

## Header File

Defined in [SDL_stdinc.h](https://github.com/libsdl-org/SDL/blob/SDL2/include/SDL_stdinc.h)

## Syntax

```c
#define SDL_MAX_UINT16  ((Uint16)0xFFFF)        /* 65535 */
```
