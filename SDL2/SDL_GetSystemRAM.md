# SDL_GetSystemRAM

Get the amount of RAM configured in the system.

## Syntax

```c
int SDL_GetSystemRAM(void);

```

## Return Value

Returns the amount of RAM configured in the system in MiB.

## Version

This function is available since SDL 2.0.1.
