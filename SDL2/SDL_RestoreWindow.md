# SDL_RestoreWindow

Restore the size and position of a minimized or maximized window.

## Syntax

```c
void SDL_RestoreWindow(SDL_Window * window);

```

## Function Parameters

|                |                       |
| -------------- | --------------------- |
| **window**     | the window to restore |

## Version

This function is available since SDL 2.0.0.
