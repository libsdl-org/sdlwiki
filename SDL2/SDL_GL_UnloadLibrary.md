# SDL_GL_UnloadLibrary

Unload the OpenGL library previously loaded by [SDL_GL_LoadLibrary](SDL_GL_LoadLibrary)().

## Syntax

```c
void SDL_GL_UnloadLibrary(void);

```

## Version

This function is available since SDL 2.0.0.
