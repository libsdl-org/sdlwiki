# SDL_HAPTIC_GAIN

Device can set global gain.

## Header File

Defined in [SDL_haptic.h](https://github.com/libsdl-org/SDL/blob/SDL2/include/SDL_haptic.h)

## Syntax

```c
#define SDL_HAPTIC_GAIN       (1u<<12)
```

## Remarks

Device supports setting the global gain.
