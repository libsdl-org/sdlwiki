# SDL_ShowWindow

Show a window.

## Syntax

```c
void SDL_ShowWindow(SDL_Window * window);

```

## Function Parameters

|                |                    |
| -------------- | ------------------ |
| **window**     | the window to show |

## Version

This function is available since SDL 2.0.0.
