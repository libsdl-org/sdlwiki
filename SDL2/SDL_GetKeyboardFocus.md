# SDL_GetKeyboardFocus

Query the window which currently has keyboard focus.

## Syntax

```c
SDL_Window * SDL_GetKeyboardFocus(void);

```

## Return Value

Returns the window with keyboard focus.

## Version

This function is available since SDL 2.0.0.
