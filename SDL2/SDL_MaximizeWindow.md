# SDL_MaximizeWindow

Make a window as large as possible.

## Syntax

```c
void SDL_MaximizeWindow(SDL_Window * window);

```

## Function Parameters

|                |                        |
| -------------- | ---------------------- |
| **window**     | the window to maximize |

## Version

This function is available since SDL 2.0.0.
