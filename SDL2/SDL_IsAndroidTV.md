# SDL_IsAndroidTV

Query if the application is running on Android TV.

## Syntax

```c
SDL_bool SDL_IsAndroidTV(void);

```

## Return Value

Returns [SDL_TRUE](SDL_TRUE) if this is Android TV, [SDL_FALSE](SDL_FALSE)
otherwise.

## Version

This function is available since SDL 2.0.8.
