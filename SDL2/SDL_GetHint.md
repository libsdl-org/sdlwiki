# SDL_GetHint

Get the value of a hint.

## Syntax

```c
const char * SDL_GetHint(const char *name);

```

## Function Parameters

|              |                   |
| ------------ | ----------------- |
| **name**     | the hint to query |

## Return Value

Returns the string value of a hint or NULL if the hint isn't set.

## Version

This function is available since SDL 2.0.0.
