# SDL_MUTEX_TIMEDOUT

Synchronization functions which can time out return this value if they time out.

## Header File

Defined in [SDL_mutex.h](https://github.com/libsdl-org/SDL/blob/SDL2/include/SDL_mutex.h)

## Syntax

```c
#define SDL_MUTEX_TIMEDOUT  1
```
