# SDL_JoystickUpdate

Update the current state of the open joysticks.

## Syntax

```c
void SDL_JoystickUpdate(void);

```

## Remarks

This is called automatically by the event loop if any joystick events are
enabled.

## Version

This function is available since SDL 2.0.0.
