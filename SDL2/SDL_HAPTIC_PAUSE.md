# SDL_HAPTIC_PAUSE

Device can be paused.

## Header File

Defined in [SDL_haptic.h](https://github.com/libsdl-org/SDL/blob/SDL2/include/SDL_haptic.h)

## Syntax

```c
#define SDL_HAPTIC_PAUSE      (1u<<15)
```

## Remarks

Devices supports being paused.
