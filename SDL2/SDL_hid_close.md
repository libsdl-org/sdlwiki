# SDL_hid_close

Close a HID device.

## Syntax

```c
void SDL_hid_close(SDL_hid_device *dev);

```

## Function Parameters

|             |                                                               |
| ----------- | ------------------------------------------------------------- |
| **dev**     | A device handle returned from [SDL_hid_open](SDL_hid_open.md)(). |

## Version

This function is available since SDL 2.0.18.
