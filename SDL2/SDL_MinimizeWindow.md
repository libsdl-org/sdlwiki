# SDL_MinimizeWindow

Minimize a window to an iconic representation.

## Syntax

```c
void SDL_MinimizeWindow(SDL_Window * window);

```

## Function Parameters

|                |                        |
| -------------- | ---------------------- |
| **window**     | the window to minimize |

## Version

This function is available since SDL 2.0.0.
