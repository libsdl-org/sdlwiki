# SDL_Vulkan_UnloadLibrary

Unload the Vulkan library previously loaded by [SDL_Vulkan_LoadLibrary](SDL_Vulkan_LoadLibrary.md)()

## Syntax

```c
void SDL_Vulkan_UnloadLibrary(void);

```

## Version

This function is available since SDL 2.0.6.
