# Mix_ResumeMusic

Resume the music stream.

## Syntax

```c
void Mix_ResumeMusic(void);

```

## Remarks

It is legal to resume an unpaused music stream; it causes no effect and
reports no error.

## Version

This function is available since SDL_mixer 2.0.0.
