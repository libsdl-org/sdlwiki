# SDL_RTF_VERSION

This is the version number macro for the current SDL_rtf version.

## Header File

Defined in SDL_rtf.h

## Syntax

```c
#define SDL_RTF_VERSION \
    SDL_VERSIONNUM(SDL_RTF_MAJOR_VERSION, SDL_RTF_MINOR_VERSION, SDL_RTF_MICRO_VERSION)
```
