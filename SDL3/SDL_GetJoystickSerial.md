# SDL_GetJoystickSerial

Get the serial number of an opened joystick, if available.

## Syntax

```c
const char * SDL_GetJoystickSerial(SDL_Joystick *joystick);

```

## Function Parameters

|                  |                                                                                       |
| ---------------- | ------------------------------------------------------------------------------------- |
| **joystick**     | the [SDL_Joystick](SDL_Joystick.md) obtained from [SDL_OpenJoystick](SDL_OpenJoystick.md)() |

## Return Value

Returns the serial number of the selected joystick, or NULL if unavailable.

## Remarks

Returns the serial number of the joystick, or NULL if it is not available.

## Version

This function is available since SDL 3.0.0.
