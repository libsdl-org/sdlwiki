# SDL_UnlockJoysticks

Unlocking for atomic access to the joystick API 

## Syntax

```c
void SDL_UnlockJoysticks(void) SDL_RELEASE(SDL_joystick_lock);

```

## Version

This function is available since SDL 3.0.0.
