# SDL_HAPTIC_RAMP

Ramp effect supported.

## Header File

Defined in [<SDL3/SDL_haptic.h>](https://github.com/libsdl-org/SDL/blob/main/include/SDL3/SDL_haptic.h)

## Syntax

```c
#define SDL_HAPTIC_RAMP         (1u<<6)
```

## Remarks

Ramp haptic effect.

## Version

This macro is available since SDL 3.0.0.
