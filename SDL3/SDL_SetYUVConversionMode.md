# SDL_SetYUVConversionMode

Set the YUV conversion mode 

## Syntax

```c
void SDL_SetYUVConversionMode(SDL_YUV_CONVERSION_MODE mode);

```

## Function Parameters

|              |                     |
| ------------ | ------------------- |
| **mode**     | YUV conversion mode |

## Version

This function is available since SDL 3.0.0.
