# SDL_CloseJoystick

Close a joystick previously opened with [SDL_OpenJoystick](SDL_OpenJoystick)().

## Syntax

```c
void SDL_CloseJoystick(SDL_Joystick *joystick);

```

## Function Parameters

|                  |                              |
| ---------------- | ---------------------------- |
| **joystick**     | The joystick device to close |

## Version

This function is available since SDL 3.0.0.
