# SDL_CloseSensor

Close a sensor previously opened with [SDL_OpenSensor](SDL_OpenSensor)().

## Syntax

```c
void SDL_CloseSensor(SDL_Sensor *sensor);

```

## Function Parameters

|                |                                              |
| -------------- | -------------------------------------------- |
| **sensor**     | The [SDL_Sensor](SDL_Sensor) object to close |

## Version

This function is available since SDL 3.0.0.
