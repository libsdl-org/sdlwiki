# SDL_GetClipboardUserdata

Retrieve previously set userdata if any.

## Syntax

```c
void* SDL_GetClipboardUserdata(void);

```

## Return Value

Returns a pointer to the data or NULL if no data exists

## Version

This function is available since SDL 3.0.0.
