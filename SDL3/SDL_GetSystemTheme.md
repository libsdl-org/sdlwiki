# SDL_GetSystemTheme

Get the current system theme 

## Syntax

```c
SDL_SystemTheme SDL_GetSystemTheme(void);

```

## Return Value

Returns the current system theme, light, dark, or unknown

## Version

This function is available since SDL 3.0.0.
