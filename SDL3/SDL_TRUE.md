# SDL_TRUE

A boolean true.

## Header File

Defined in [SDL_stdinc.h](https://github.com/libsdl-org/SDL/blob/main/include/SDL3/SDL_stdinc.h), but apps should use `#include <SDL3/SDL.h>`

## Syntax

```c
#define SDL_TRUE 1
```

## Version

This macro is available since SDL 3.0.0.
