# SDL_TextInputActive

Check whether or not Unicode text input events are enabled.

## Syntax

```c
SDL_bool SDL_TextInputActive(void);

```

## Return Value

Returns [SDL_TRUE](SDL_TRUE) if text input events are enabled else
[SDL_FALSE](SDL_FALSE).

## Version

This function is available since SDL 3.0.0.
