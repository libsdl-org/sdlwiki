# SDL_Metal_GetLayer

Get a pointer to the backing CAMetalLayer for the given view.

## Syntax

```c
void* SDL_Metal_GetLayer(SDL_MetalView view);

```

## Function Parameters

|              |                                           |
| ------------ | ----------------------------------------- |
| **view**     | the [SDL_MetalView](SDL_MetalView.md) object |

## Return Value

Returns a pointer

## Version

This function is available since SDL 3.0.0.
