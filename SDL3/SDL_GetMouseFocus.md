# SDL_GetMouseFocus

Get the window which currently has mouse focus.

## Syntax

```c
SDL_Window * SDL_GetMouseFocus(void);

```

## Return Value

Returns the window with mouse focus.

## Version

This function is available since SDL 3.0.0.
