# SDL_GetYUVConversionMode

Get the YUV conversion mode

## Syntax

```c
SDL_YUV_CONVERSION_MODE SDL_GetYUVConversionMode(void);

```

## Return Value

Returns YUV conversion mode

## Version

This function is available since SDL 3.0.0.
