# SDL_GetNumGamepadTouchpads

Get the number of touchpads on a gamepad.

## Syntax

```c
int SDL_GetNumGamepadTouchpads(SDL_Gamepad *gamepad);

```

## Function Parameters

|                 |           |
| --------------- | --------- |
| **gamepad**     | a gamepad |

## Return Value

Returns number of touchpads

## Version

This function is available since SDL 3.0.0.
