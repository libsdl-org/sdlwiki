# SDL_PLATFORM_DOS

A preprocessor macro that is only defined if compiling for MS-DOS.

## Header File

Defined in [<SDL3/SDL_platform_defines.h>](https://github.com/libsdl-org/SDL/blob/main/include/SDL3/SDL_platform_defines.h)

## Syntax

```c
#define SDL_PLATFORM_DOS 1
```

## Version

This macro is available since SDL 3.6.0.
