# SDL_HideCursor

Hide the cursor.

## Syntax

```c
int SDL_HideCursor(void);

```

## Return Value

Returns 0 on success or a negative error code on failure; call
[SDL_GetError](SDL_GetError.md)() for more information.

## Version

This function is available since SDL 3.0.0.
