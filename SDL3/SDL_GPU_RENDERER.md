# SDL_GPU_RENDERER

The name of the GPU renderer.

## Header File

Defined in [<SDL3/SDL_render.h>](https://github.com/libsdl-org/SDL/blob/main/include/SDL3/SDL_render.h)

## Syntax

```c
#define SDL_GPU_RENDERER        "gpu"
```

## Version

This macro is available since SDL 3.4.0.
