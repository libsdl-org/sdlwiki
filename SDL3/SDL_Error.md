# SDL_Error

[SDL_Error](SDL_Error)() 

## Syntax

```c
int SDL_Error(SDL_errorcode code);

```

## Function Parameters

|              |            |
| ------------ | ---------- |
| **code**     | Error code |

## Return Value

Returns unconditionally -1.

## Version

This function is available since SDL 3.0.0.
