# TTF_Version

This function gets the version of the dynamically linked SDL_ttf library.

## Header File

Defined in SDL_ttf.h

## Syntax

```c
int TTF_Version(void);

```

## Return Value

Returns SDL_ttf version

## Version

This function is available since SDL_ttf 3.0.0.
